## Descrição do Projeto

CRUD com NestJS, Mysql e Documentação Swagger.

## Desenvolvimento

```bash
Siga o Passo-a-Passo informado no arquivo raiz de nome: Procedimento.MD para desenvolver sua api.
```

## Instalação

```bash
$ npm install
```

## Como rodar o BackEnd

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
